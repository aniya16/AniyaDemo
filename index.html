Welcome to Web Technologies

This website introduces the core building blocks of the web: HTML, CSS, and JavaScript. These technologies allow developers to structure content, style webpages, and make them interactive.

What is the Internet?

The Internet is the backbone of the Web, the technical infrastructure that makes the Web possible, enabling data to travel between servers and browsers so users can access websites, applications, and online services from anywhere in the world.

What is the Web?

A web based application is software that runs in a browser rather than requiring installation on a device, but users can still interact with it. For example, BBC, Reuters, or other interactive news platforms allow users to browse articles, watch videos, customise settings, and engage with dynamic content
